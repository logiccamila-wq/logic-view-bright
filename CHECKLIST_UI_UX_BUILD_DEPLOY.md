# Checklist UI/UX, Build e Deploy para Teste

## UI/UX
- [ ] Prototipação (Figma, Wireframe ou similar)
- [ ] Aprovação do layout com stakeholders
- [ ] Implementação dos componentes (React, CSS/Tailwind)
- [ ] Testes de usabilidade com usuários reais
- [ ] Ajustes de acessibilidade (contraste, navegação por teclado, ARIA)
- [ ] Responsividade (mobile, tablet, desktop)
- [ ] Teste cross-browser (Chrome, Firefox, Edge, Safari)
- [ ] Ajustes finais de UI/UX

## Build e Deploy para Teste
- [ ] Rodar `npm run build` sem erros
- [ ] Rodar `npm run start:prod` localmente e validar funcionamento
- [ ] Deploy automático para ambiente de teste (Vercel, Netlify, etc.)
- [ ] Validar URL de preview/teste
- [ ] Testes de navegação, performance e bugs no ambiente de teste
- [ ] Coletar feedback do time/usuários
- [ ] Corrigir eventuais problemas antes do go-live

---
Este checklist garante que a experiência do usuário e a qualidade técnica estejam validadas antes do go-live.
